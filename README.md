# PDF Merge

A free, 100% client-side tool to mail-merge structured CSV data into a PDF template (e.g. for generating certificates, letters, or custom forms).

## How it works

### What you need:

- A data table in CSV format (`.csv`)
- A template document in PDF format (`.pdf`)

### Steps:

1. **Upload** your PDF template and CSV data.
2. **Add fields** to the preview canvas by dragging CSV column headers.
3. **Style fields** by adjusting font family, size, color, and alignment in the editor panel.
4. **Export** either as individual PDFs bundled in a ZIP file, or as a single combined PDF.

---

## Local Development

### Prerequisites

- Node.js ≥ 18

### Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
